fix(core): 메모리 누수 수정 및 리소스 정리 로직 강화 (v1.1.1)

서버 안정성 향상을 위해 주요 서비스의 메모리 누수 및 리소스 관리 문제를 수정하고 최적화함.

- **메모리 누수 수정**
  - **SMS**: RxJS `take(1)`을 사용하여 HTTP 요청 완료 후 구독 자동 해제 적용
  - **MinIO**: 객체 목록 조회 스트림 완료 시 명시적 `destroy()` 호출로 리소스 누수 방지

- **성능 및 리소스 최적화**
  - **File**: `LogCleanupService` 및 `LogFileService`의 배열 순회 로직을 단일 루프로 통합하여 메모리 사용량 개선
  - **SMB**: `finally` 블록 내 리소스 정리 로직을 개별 `try-catch`로 분리하여 안전한 연결 해제 보장
  - **WishUniv**: 희망대학 목록 처리 시 불필요한 배열 생성 및 `.flat()` 중복 호출 제거

- **문서 업데이트**
  - `package.json` 버전을 1.1.1로 올림
  - `CHANGELOG.md` 및 `README.md`에 테스트 프레임워크 변경 및 수정 사항 반영
