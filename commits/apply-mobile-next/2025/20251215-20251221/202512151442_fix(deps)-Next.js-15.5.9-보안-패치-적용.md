## 업그레이드 사유

React Server Components 관련 심각한 보안 취약점 대응:

- CVE-2025-66478 (CVSS 10.0): 원격 코드 실행(RCE) 취약점
- CVE-2025-55183/55184: 악의적 HTTP 요청으로 인한 DoS 취약점
- CVE-2025-67779: 이전 패치 불완전 수정 보완

## 영향 범위

- App Router 사용하는 Next.js 15.x~16.x 버전 영향
- 워크어라운드 없음, 업그레이드만이 유일한 해결책
