fix(clarity): 광고 차단기 에러 필터링 스크립트 추가

- 광고 차단기(AdBlock)로 인해 발생하는 스크립트 에러가 Microsoft Clarity에 불필요하게 기록되는 문제를 해결함.
- GTM/Clarity 로드 전에 실행되는 에러 필터링 스크립트(`ad-error-filter.ts`)를 `<head>` 태그에 추가함.
- 광고 관련 URL 패턴을 감지하여 해당 에러의 전파를 차단하고, 기존 에러 핸들러는 정상적으로 동작하도록 보존함.
- 스크립트 로직을 `shared/scripts` 폴더에 별도 모듈로 분리하여 프로젝트 구조 패턴을 준수함.
