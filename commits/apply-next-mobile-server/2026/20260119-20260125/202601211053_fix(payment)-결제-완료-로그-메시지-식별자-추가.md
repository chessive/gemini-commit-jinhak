fix(payment): 결제 완료 로그 메시지 모바일 식별자 추가 및 은행 코드 수정

- 결제 완료 시 기록되는 로그 데이터(`RetMsg`)에 'Mobile/' 접두사를 추가하여 모바일 환경에서의 접수임을 명확히 식별할 수 있도록 개선함.
  - `returnMsg`: '접수성공' -> 'Mobile/접수성공'
  - `returnMsg` (CompleteApply): '접수성공' -> 'Mobile/접수성공-CompleteApply'
- `bankCode`: 기존에 하드코딩된 'mobile' 값을 빈 문자열('')로 수정하여 데이터 정합성을 확보함.
