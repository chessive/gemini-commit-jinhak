feat(excel): 통계 엑셀에 동적 커스텀 컬럼 기능 추가

통계 정보의 '학과별 통계' 엑셀 시트에 `CustomConfig`를 통해 동적으로 열(Column)을 추가하고 정렬할 수 있는 기능을 구현함. 이를 통해 웹 관리자에서 설정한 커스텀 필드를 엑셀에 즉시 반영할 수 있어 확장성과 유연성을 크게 향상시킴.

### 주요 변경 사항

- **1. `CustomConfig` 모델 확장 (`data_config.dart`):**
  - `data.statisticsExcel.majorSheet.customColumns` 경로에 설정을 추가함.
  - `enabled` (boolean)와 `columns` (list) 속성을 포함하는 `DataConfigStatisticsExcel...` 관련 모델 클래스들을 신설하여 `CustomConfig` 스키마를 확장함.
  - `columns` 리스트의 각 항목은 `index`, `label`, `value`, `width`를 포함.

- **2. 동적 헤더 생성 및 정렬 (`excel_service.dart`):**
  - `_createMajorHeaderLabels` 메서드에서 `CustomConfig` 설정을 읽어 기본 4개 열(idx, 전형명, 모집단위명, 조회수) 외에 추가 열들을 동적으로 생성하도록 수정함.
  - `_ExcelHeaderLabel` 모델에 `position` 필드를 추가하고, 모든 열을 `position` 값 기준으로 정렬하여 최종 열 순서를 결정하도록 구현함. (기본 열은 10단위 `position` 가짐)

- **3. 동적 데이터 조회 로직 구현 (`excel_service.dart`):**
  - `_getMajorCellValue` 메서드의 `default` 케이스에서, `ApplyMajor.toJson()`을 통해 모델을 JSON으로 변환하고 `CustomConfig`에 정의된 `value` 키를 사용해 필드 값을 동적으로 조회하도록 구현함.
  - 조회된 값이 `num` 타입일 경우 `double`로 자동 변환하고, 존재하지 않거나 오류 발생 시에는 빈 문자열을 반환하여 안정성을 확보함.

- **4. 너비 자동 보정:**
  - `CustomConfig`에서 `width` 값이 1 미만일 경우, 이를 비율로 간주하여 100을 곱한 값으로 자동 보정하는 로직을 추가함 (예: `0.14` -> `14.0`).

### 기대 효과

- **유연성 및 확장성 증대:** 앱 재배포 없이 웹 관리자에서 `CustomConfig` 설정만으로 엑셀에 '모집인원', '경쟁률' 등 원하는 통계 정보를 자유롭게 추가/제거/정렬할 수 있음.
- **유지보수성 향상:** 하드코딩된 열 구조에서 벗어나, 설정 기반의 동적 구조로 변경되어 향후 새로운 통계 필드 추가 시 코드 수정이 불필요해짐.
- **사용자 맞춤 리포트:** 서비스별, 시점별 요구사항에 맞춰 필요한 정보만으로 구성된 맞춤형 엑셀 리포트 생성이 가능해짐.
