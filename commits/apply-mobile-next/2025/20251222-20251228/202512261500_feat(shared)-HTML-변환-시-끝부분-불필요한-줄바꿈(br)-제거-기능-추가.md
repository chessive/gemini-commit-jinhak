feat(shared): HTML 변환 시 끝부분 불필요한 줄바꿈(<br>) 제거 기능 추가

- `convertHtmlToReactElements` 유틸리티에서 HTML 변환 결과의 마지막에 위치한 연속된 `<br>` 태그를 자동으로 제거하는 기능을 추가함.
- **주요 변경 사항**:
  - `trimTrailingBr` 함수를 구현하여 React 노드 배열 끝부분의 `<br>` 요소를 `trim()`처럼 정리함.
  - 변환 파이프라인의 마지막 단계(결과 래핑 전)에 해당 정제 로직을 적용하여 불필요한 하단 여백이 생기는 문제를 방지함.
