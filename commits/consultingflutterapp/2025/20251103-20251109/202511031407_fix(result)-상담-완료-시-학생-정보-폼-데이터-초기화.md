fix(result): 상담 완료 시 학생 정보 폼 데이터 초기화

상담 결과 화면(ResultScreen, ResultDetailScreen)에서 '상담완료' 버튼을 클릭했을 때, 학생 정보 입력 폼 데이터가 초기화되지 않는 버그를 수정함.

### 주요 변경 사항

- **`TopCard.dart` (결과 화면):**
  - `_onConsultFinishPressed` 메서드에 `ConsultingStore`의 `resetFormData()` 호출을 추가하여, 상담 종료 시 학생 정보 폼 데이터를 초기화하도록 함.

- **`DetailTopCard.dart` (상세 결과 화면):**
  - '상담완료' 버튼의 `onPressed` 핸들러에 `resetFormData()` 호출을 추가하여 동일한 초기화 로직을 적용함.

### 해결된 문제

- **데이터 무결성:** 새로운 상담을 시작할 때 이전 학생의 정보가 폼에 남아있던 문제를 해결.
- **일관성 확보:** `StudentInfoScreen`의 '이전' 버튼과 동일한 데이터 초기화 동작을 보장하여 사용자 플로우 전반의 일관성을 높임.
