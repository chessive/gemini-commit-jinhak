fix(html-rendering): HTML 테이블 rowspan 세로 정렬 및 줄 간격 개선

Flutter 앱에서 HTML 테이블 렌더링 시 `rowspan` 속성을 사용한 셀의 세로 정렬 문제와 테이블 셀 내부 줄 간격 부족 문제를 해결하고, 코드 품질을 개선함.

### 주요 변경 사항

- **`rowspan` 셀 세로 중앙 정렬 (`convert_html.dart`):**
  - `<th>` 태그에 `height: 100%` 속성을 복원하고, `<th>`, `<td>` 태그에 `vertical-align: middle`을 추가하여 `rowspan`이 적용된 셀의 텍스트가 세로 중앙에 정렬되도록 수정함.
  - `isCalculate` 여부와 관계없이 `height: 100%`가 적용되도록 하여 일관된 동작을 보장함.

- **테이블 셀 내부 줄 간격 개선 (`convert_html.dart`):**
  - 테이블 셀 내부의 `<p>` 태그 중 첫 번째 `<p>`를 제외한 나머지 `<p>` 태그에 `margin-top: 0.5em`을 추가하여 여러 줄 텍스트의 가독성을 향상시킴.

- **코드 품질 개선:**
  - **HTML 스타일 상수화**: `TABLE_PARAGRAPH_SPACING`, `TABLE_CELL_LINE_HEIGHT`, `TABLE_CELL_HEIGHT_FULL` 상수를 도입하여 매직 넘버를 제거하고 유지보수성을 높임.
  - **성능 최적화**: `<p>` 태그의 부모 DOM 탐색 로직을 `querySelectorAll` 대신 `children` 순회를 통해 최적화하여 렌더링 성능을 향상시킴.
  - **에러 핸들링 추가**: DOM 조작 실패 시 `try-catch` 블록으로 예외 처리를 추가하여 앱의 견고성을 확보함.

### 개선 효과

- **UI 정확성**: HTML 테이블의 `rowspan` 셀이 의도대로 세로 중앙에 정렬되어 시각적 완성도를 높임.
- **가독성 향상**: 테이블 셀 내부의 여러 줄 텍스트 간격이 적절히 확보되어 정보 전달력을 높임.
- **유지보수성 증대**: 스타일 상수를 통해 중앙 집중식 관리가 가능해지고, 코드 변경 시 영향 범위를 최소화함.
- **성능 최적화**: 불필요한 DOM 탐색을 줄여 렌더링 속도를 개선함.
- **안정성 강화**: 예외 처리 로직을 추가하여 예상치 못한 오류 발생 시에도 앱이 안정적으로 동작하도록 함.
