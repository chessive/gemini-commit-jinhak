feat(common): 공용 UI 컴포넌트 (FormDrawer, ConfirmDialog, JsonEditor) 구현

계산식 설정 페이지 및 향후 관리자 페이지에서 재사용될 수 있는 공용 UI 컴포넌트들을 구현함.

### 주요 변경 사항

- **`ConfirmDialog` 구현 (`confirm-dialog.tsx`):**
  - 사용자에게 중요한 작업을 수행하기 전에 확인을 요청하는 공용 Dialog 컴포넌트.
  - `title`, `message`, `onConfirm`, `onCancel` 등 필수 속성을 포함.
- **`FormDrawer` 구현 (`form-drawer.tsx`):**
  - 폼 입력을 위한 공용 Drawer 컴포넌트.
  - 우측에서 슬라이드인되며, 제목, 닫기 버튼, 제출/취소 버튼 및 콘텐츠 영역으로 구성.
  - `isSubmitting` 상태를 통해 버튼 비활성화 기능 제공.
- **`JsonEditor` 구현 (`json-editor.tsx`):**
  - JSON 데이터를 시각적으로 편집하고 프리뷰할 수 있는 컴포넌트.
  - `react-monaco-editor`를 동적으로 임포트하여 JSON 하이라이팅, 포맷팅, 읽기 전용 모드 등을 지원.
  - SSR 환경에서도 안전하게 로드되도록 `next/dynamic` 사용.
