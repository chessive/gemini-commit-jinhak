chore(release): v1.1.5 버전 업데이트 및 변경 이력 반영

- 프로젝트 버전을 1.1.3에서 v1.1.5로 업데이트하고, 최근 추가된 주요 기능 및 개선 사항들을 `CHANGELOG.md`에 기록함.
- **v1.1.5 주요 업데이트 요약:**
  - **신규 기능**: UnivFilter 배열 기반 시간 제한 필터링 구현.
  - **모듈 통합**: 배너 캐시 중앙 관리 시스템 통합 및 내부 API 연동.
  - **문서 표준화**: 전 모듈 Swagger 문서 상수화 및 응답 예시(Examples) 분리.
  - **보안 강화**: InternalApiGuard 타이밍 공격 방지 로직 적용.
  - **인프라**: Dockerfile 작업 디렉토리 경로 정규화.
  - **버그 수정**: SMB 파일 목록 필드 누락 및 결제 로그 식별자 오류 수정.
