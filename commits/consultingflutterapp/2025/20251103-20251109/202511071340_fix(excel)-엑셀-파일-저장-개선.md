fix(excel): 엑셀 파일 저장 기능 안정성 및 편의성 개선

엑셀 파일 저장 시 발생할 수 있는 데이터 유실 문제를 방지하고 사용자 편의성을 개선함.

- **파일명 고유성 강화**:
  - 엑셀 파일 저장 시, 파일명에 날짜(YYYYMMDD)뿐만 아니라 시간(HHmm)까지 포함하도록 변경함.
  - 하루에 여러 번 파일을 저장해도 기존 파일이 덮어쓰여지는 문제를 방지하고 데이터 유실 위험을 줄임.

- **파일 확장자 자동 추가**:
  - 파일 저장 대화상자에서 사용자가 파일명을 입력할 때 `.xlsx` 확장자를 생략하더라도, 시스템이 자동으로 확장자를 추가하도록 수정함.
  - 사용자가 확장자를 누락하여 파일이 잘못 저장되는 실수를 방지하고 편의성을 개선함.
