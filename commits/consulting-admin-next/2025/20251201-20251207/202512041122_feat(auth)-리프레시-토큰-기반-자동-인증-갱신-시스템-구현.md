feat(auth): 리프레시 토큰 기반 자동 인증 갱신 시스템 구현

- Access Token을 sessionStorage 대신 인메모리(authToken.ts)로 관리하여 XSS 공격 방어 기능 강화.
- axios 인터셉터에 401 에러 발생 시 HttpOnly Refresh Token을 사용하여 Access Token을 자동으로 갱신하는 로직 구현.
- 토큰 갱신 중 발생한 요청들을 큐잉하여 갱신 완료 후 자동으로 재처리하도록 구현.
- 로그아웃 시 서버에 Refresh Token 무효화를 요청하고 클라이언트 인메모리 Access Token을 초기화하는 로직 추가.
- 초기 페이지 로드 시 Refresh Token을 통한 자동 인증 복구 로직 구현.
- 관련 API (sign-out.ts, refresh-token.ts) 및 훅 (use-sign-in-mutation.ts, user-context.tsx, user-toggler.tsx) 업데이트.
- API 응답 transformResponse에서 에러 응답을 직접 처리하지 않고 인터셉터에 위임하도록 수정.
